What is Networking?

Networking is the practice or system of meeting or talking to other people. Networking is the study of the methods that are used to create a network in which data can be transferred from one node to another either wired or wirelessly. Networking is a vast field to learn.

Topologies

Topology is a structure in which the network is formed. Network speed and efficiency can vary according to the topology. There are different types of topologies. Some of them are explained below:

  • Bus Topology
  • Star Topology
  • Ring Topology
  • Tree Topology
  • Mess Topology
  • Hybrid Topology

Bus Topology

Bus Topology is a simple and specific kind of network topology in which all the nodes are connected to each other through a backbone cable to transfer data.

Star Topology

In Star Topology, all the devices/nodes are connected to a central node e.g., switch. Here, if one or more nodes get damaged, the network will be the same as it was before.

Ring Topology

Ring Topology is collision free topology in which one node is connected to its two contiguous nodes. If a single node gets damaged, whole network will go down.

Tree Topology

Tree Topology is a topology in which all the nodes are connected in a tree type structure. This structure contains a main 0 level node, if that's get damaged, whole network will go down.

Mess Topology

Mess Topology is the most reliable and expensive topology out there. In Mess Topology, all of the nodes/devices are connected to each other in order to transmit data. If one or more nodes get damaged, network will not be affected instead it will be same as before.

Hybrid Topology

Hybrid Topology is a combination of more than one topologies that we discussed above. It is made by combining more than one type of topology e.g., bus, star, tree etc.

Networking Cables

When we talk about physical networking, networking cables perform a fundamental role. Networking cables comes in various varients but the three most popular networking cables are explained below:

  • Coaxial Cable
  • Twisted pair Cable
  • Optical Fibre Cable

Coaxial Cable

Coaxial cable is a cheap cable among other networking cables. It has a copper wire in the middle that is responsible for transmitting data from one device to another. An aluminium layer is present above the core copper wire that protects the cable from collision of data.

Twisted Pair Cable

A twisted pair cable comprises of two separate insulated copper wires, which are twisted together and run in parallel. The copper wires are typically 1mm in diameter. One of the wires is used to transmit data and the other is the ground reference.

Optical Fibre Cable

A fiber-optic cable, also known as an optical-fiber cable, is an expensive cable that is used for heigher bandwidth. Optical Fibre Cables use light reflection technology to transmit data from one place to another. The optical fiber elements are typically individually coated with plastic layers and contained in a protective tube suitable for the environment where the cable is used.

OSI Model

OSI Model stands for Open Systems Interconnection Model. It is developed by ISO (International Organization of Standardization). It is a conceptual networking model that is used to describe the functions of the networking system. It consists of 7 layers:

  1. Application Layer
  2. Presentation Layer
  3. Session Layer
  4. Transport layer
  5. Network Layer
  6. Data Link Layer
  7. Physical Layer

1. Plysical Layer

It is the lowest layer in OSI model. It is responsible for all the physical transmissions. It transmits data across the network. Hub, repeaters, cabling are physical layer devices.

2. Data Link Layer

Data link layer is responsible for node-to-node data transfer. It transmits data in frames. The data link layer also corrects errors that may have occurred at the physical layer. Data link layer comprises of two sub-layers:

  • MAC (Media Access Control)
  • LLC (Logical Link Control)

3. Network Layer

The network layer is responsible for receiving frames from the data link layer, and delivering them to their intended destinations based on the addresses contained inside the frame. Network layer uses IP address to find the data destination.

4. Transport Layer

The transport layer manages the delivery and error checking of data packets. It regulates the size, sequencing, and ultimately the transfer of data between systems and hosts. One of the most common examples of the transport layer is TCP or the Transmission Control Protocol.

5. Session Layer

The session layer controls the conversations between different computers. A session or connection between machines is set up, managed, and termined at layer 5. Session layer services also include authentication and reconnections.

6. Presentation Layer

The presentation layer formats or translates data for the application layer based on the syntax or semantics that the application accepts. Because of this, it at times also called the syntax layer. This layer can also handle the encryption and decryption required by the application layer.

7. Application Layer

At this layer, both the end user and the application layer interact directly with the software application. The application layer identifies communication partners, resource availability, and synchronizes communication.

LAN, MAN and WAN

LAN (Local Area Network)

LAN stands for Local Area Network. It is a network that covers less geographical area in comparision to MAN and WAN. LAN is used for home networks, office networks etc. It can be wired or wireless.

MAN (Matropolitan Area Network)

A MAN stands for Matropolitan Area Network, is a larger network that usually spans several buildings in the same city or town. It is also used to establish network across the cities.

WAN (Wide Area Network)

A WAN stands for Wide Area Network, in comparison to a MAN, is not restricted to a geographical location, although it might be confined within the bounds of a state or country. A WAN connects several LANs, and may be limited to an enterprise (a corporation or an organization) or accessible to the public. The technology is high speed and relatively expensive. The Internet is an example of a WAN.

IP Address

An Internet Protocol address is a numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication.

MAC Address

A media access control address is a unique identifier assigned to a network interface controller or a network device for use as a network address in communications within a network segment.

Network Interface Card (NIC)

NIC is an acronym for Network interface controller or Network Interface Card, but it can also be known as a network adapter or LAN network. It is a piece of computer hardware, used for connecting a computer to a network, either through cables or wirelessly. A laptop with both wired and wireless connections, contains two MAC addresses.
